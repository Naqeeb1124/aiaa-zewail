# Color Palette

## Brand Colors
- **Deep Blue**: `#2b4b77` (Primary)
- **Dark Blue**: `#1a3d6d` (Backgrounds / Dark Mode)
- **Green**: `#78af03` (Secondary / Success)
- **Cyan**: `#00b4d1` (Accent / Buttons)

## Usage Guidelines
- **Deep Blue** (`#2b4b77`): Use for primary elements, cards, and headers.
- **Dark Blue** (`#1a3d6d`): Use for the main page background (Dark Mode base).
- **Green** (`#78af03`): Use for success states and secondary accents.
- **Cyan** (`#00b4d1`): Use for primary buttons, links, and highlights.
