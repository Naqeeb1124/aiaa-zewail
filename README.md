This repo is a minimal fullstack Next.js app configured for Vercel + Firebase (Auth + Firestore + Storage).
It includes pages and an admin skeleton.

How to run locally:
1. Install dependencies: npm install
2. Add environment variables (see .env.example)
3. Run: npm run dev

Deploy:
- Push to GitHub and deploy on Vercel. Add NEXT_PUBLIC_... Firebase vars in Vercel.
